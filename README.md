# 🏨 BRAYAN'S HOT HOTEL

A polished, high-quality first-person 3D horror survival game.

## 🎮 Game Features

- **Full Exploration**: Navigate a large, multi-floor hotel with realistic layout
- **First-Person Gameplay**: Full WASD movement with mouse-look controls
- **Intelligent Enemies**: 4 unique AI-controlled enemies with distinct behaviors
- **Dynamic Objectives**: Randomized objectives every playthrough
- **Atmospheric Horror**: Horror survival experience with strong immersion
- **Security Systems**: Cameras and tracking systems to aid survival
- **Progressive Difficulty**: Enemies become smarter and more aggressive over time

## 🎯 Objectives

Every playthrough includes randomized objectives:
- Repair Generator
- Restore Security Network
- Recover Hotel Records
- Investigate VIP Suites
- Collect Keycards
- Repair Elevators
- Restore Server Systems
- Send Emergency Broadcast
- Unlock Exit Systems

## 👾 Enemies

### Abhi - "The Technician"
Disables cameras and sabotages tracking systems. Frequently visits maintenance areas.

### Asmita - "The Broadcaster"
Hijacks the hotel PA system, gives false announcements, and plays psychological tricks.

### Sri - "The Trickster"
Creates false tracker signals and appears in multiple locations. Uses deception over speed.

### Brayan - "The Pursuer"
Stays outside the hotel but becomes active once the player escapes. The final threat.

## 🎮 Controls

| Key | Action |
|-----|--------|
| **W/A/S/D** | Move Forward/Left/Backward/Right |
| **Mouse** | Look Around |
| **Space** | Jump |
| **Shift** | Sprint |
| **Ctrl** | Crouch |
| **F** | Toggle Flashlight |
| **E** | Interact with Objects |
| **C** | Security Cameras (in Security Room) |
| **ESC** | Pause Game |

## 🚀 How to Play

1. Open `index.html` in your web browser
2. Click "START GAME"
3. Move with WASD and look around with mouse
4. Press F to toggle flashlight
5. Explore and complete objectives
6. Avoid the 4 enemies!
